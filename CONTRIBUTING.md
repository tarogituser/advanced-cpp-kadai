# Contributing to UniDx / コントリビューションについて
  
UniDx への参加に興味をお持ちいただき、ありがとうございます。

This project welcomes contributions.  
Japanese is totally fine. English is also welcome.

---

## 🧩 目的 / Project purpose

UniDx は「学習用の DirectX フレームワーク」です。

- DirectX 入門（特にUnity経験者）
- ゲームフレームワークの構造理解に
- フレームワーク作成の参考・たたき台として

---

## 🐛 Issues（イシュー）について

Issues は以下の用途に使ってください。

- バグ報告
- 質問
- 改善提案
- ドキュメント修正提案

Please feel free to open issues in **Japanese or English**.

返信は早くない場合がありますが、ご了承ください。

---

## 🔧 Pull Requests（プルリクエスト）

Pull Request を歓迎します。

基本的な流れ：

1. Issue を作成（必須ではありません）
2. Fork する
3. ブランチを作成
4. 変更を加える
5. Pull Request を送信

Small changes are welcome (typos, comments, samples, etc.)

---

## 🎯 優先順位 / Priority

UniDx では以下を優先します。

- 利用しやすさ
- 理解しやすさ
- シンプルな実装

最適化や高度な機能よりも  
**「読みやすさ・学びやすさ」を優先**します。

---

## 📝 Coding style

- コメントは日本語でも英語でも可
- 学習者が読めるコードを意識してください
- 過度なテンプレート・メタプログラミングは避けてください
